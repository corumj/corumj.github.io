Welcome to Peripatetic.me

Date [ Fri, 23 Oct 2020 00:00:00 UTC ] Categories [ Beginnings ] Tags [ Welcome ]

So I decided to actually use this domain I’ve had for years for a website. My friend Damien told me about keeping a “TIL” journal, basically a searchable list of snippets of things I have learned that I think are interesting or might be useful in the future. So this is mostly for me, but if someone else can get some benefit from these random bits of knowledge that’s a bonus!
